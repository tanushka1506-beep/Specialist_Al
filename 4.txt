📚 Урок 4: Квадратные уравнения (ОГЭ задания 13,21)

🔹 Теория:
1. Стандартный вид: ax² + bx + c = 0
2. Дискриминант: D = b² - 4ac
3. Формула корней: x = (-b ± √D)/(2a)

📌 Пример решения:
Задача: 2x² - 5x - 3 = 0
Решение:
1) D = (-5)² - 4*2*(-3) = 25 + 24 = 49
2) x = (5 ± √49)/4 = (5 ± 7)/4
3) Ответ: x₁ = 3; x₂ = -0.5

📝 Практические задания:
1. 3x² + 8x - 11 = 0_1_-3.67_1.0_2.33_1
2. x² - 6x + 9 = 0_3_6_-3_0_0
3. 5x² - 2x - 7 = 0_1.4_-1.0_1.0_-1.4_1

💡 Полезные советы:
1. При D < 0 - корней нет
2. Если b чётное: D/4 = (b/2)² - ac
3. Проверяй знаки коэффициентов!

⚠️ Частые ошибки:
1. Потеря минуса перед b
2. Неправильный расчёт D
3. Несокращённые дроби в ответе

📌 Дополнительно:
* Для продвинутых: x² + px + 9 = 0 при D=0 → p=±6
